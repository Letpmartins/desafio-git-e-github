<h1>Desafio Git-Github Comunidade Idev</h1>

<p>Dupla: Diogo Costa e Letícia Martins</p>

<p> 
    O desafio consistiu em construir um repositório remoto no github,
    para praticar o trabalho em grupo para controle de versões em aplicações que são desenvolvidas em grupo ou dupla.
</p>

<p>
    Passos realizados durante o desafio.
</p>

<ul>
    <li>Criar o repositório remoto no github.</li>
    <li>Adicionar a licença ao repositório que nesse caso utilizamos a licença do MIT.</li>
    <li>Criar o arquivo index.html.</li>
    <li>Realizar os pushs  para as atualizações através de commits das atividades executas no arquivo.</li>
    <li>E por fim publicar a página no github pages onde você pode conferir o resultado no link abaixo.</li>

     <a href="https://letpmartins.github.io/desafio-git-e-github/"> GitHub Pages</a>
</ul>
